 CarbonSense – Data-Driven Carbon Footprint Analysis System

🌱 Overview

CarbonSense is a web-based application developed as a **Bachelor of Computer Applications (BCA)** academic project.
The application helps users calculate and analyze their carbon footprint based on daily activities such as electricity consumption, transportation distance, and fuel usage.
It uses predefined carbon emission factors to perform accurate, formula-based calculations and provides interactive visualizations and sustainability recommendations to 
encourage environmentally responsible decisions.

---

🎯 Project Objective

The primary objective of CarbonSense is to raise awareness about carbon emissions and promote sustainable living by enabling users to:

* Calculate their carbon footprint accurately.
* Understand the contribution of different activities to total emissions.
* Visualize emission data through interactive charts.
* Receive eco-friendly recommendations based on emission levels.

 🚀 How It Works

1. Users create an account or log in.
2. Enter electricity usage, transportation distance, and fuel consumption.
3. The system calculates carbon emissions using predefined emission factors.
4. Emission data is stored in the database.
5. Users can view interactive charts and analytics on the dashboard.
6. Based on the emission level (Low, Medium, High), the system provides eco-friendly recommendations.

---


 📊 Dashboard

The analytics dashboard provides:

* Total Carbon Emission
* Electricity Emission
* Transportation Emission
* Fuel Emission
* Interactive Charts
* Emission Breakdown

## 🌿 Sustainability Suggestions

Based on the calculated emission level, CarbonSense provides recommendations such as:

* Use public transportation whenever possible.
* Switch to energy-efficient appliances.
* Reduce unnecessary electricity consumption.
* Walk or cycle for short distances.
* Practice carpooling.
* Support renewable energy sources.
* Plant trees and promote environmental awareness.

💻 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/CarbonSense.git
```

Navigate to the project directory:

```bash
cd CarbonSense
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
